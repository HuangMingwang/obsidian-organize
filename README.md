# obsidian-organize

Organize Obsidian notes with auto-categorization into PARA folders, frontmatter generation, content simplification, bidirectional linking, and MOC index updates. Triggers on: organize notes, clean up vault, generate note, update MOC, add links, simplify note, categorize note, 整理笔记, 生成笔记, 精简笔记

## Install

```bash
npx skills add https://github.com/HuangMingwang/obsidian-organize
```

If this repository contains more than one skill in the future, install this specific skill with:

```bash
npx skills add https://github.com/HuangMingwang/obsidian-organize --skill obsidian-organize
```
